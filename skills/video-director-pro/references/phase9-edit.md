# Phase 9：视频剪辑与成片输出（video-edit / final-edit）

## 核心定位
最终交付环节，将所有 Clip 按分镜顺序拼接为完整成片，支持智能配乐、节奏卡点、统一调色，输出可直接使用的最终视频。

> 🔧 **本机执行模式**：不真的调剪辑工具，本阶段你产出**「剪辑交接单」**——明确的片段拼接顺序、配乐参数、调色/卡点建议，让用户在目标平台/剪映按单执行。本机能力到「给齐参数 + 顺序核对」为止。

## 前置准入铁律
- ✅ 所有 Clip 视频片段都已生成成功、无缺失、无失败；
- ✅ 所有 Clip 内容、时长都已用户确认；
- ✅ Clip 顺序已与分镜表核对无误；
- ✅ **每个 Clip 已过闸 10 成片逐帧视觉验收**（`PASS` / `ACCEPTED_WITH_RISK` 已记录风险）；任一 `FAIL_BLOCKED` 或 `AUDIT_INCOMPLETE` 的 Clip **不进剪辑**，先返修（见 `phase9-frame-audit.md`）。

## 执行流程

### Step0 成片逐帧视觉验收（闸 10，进剪辑前必过）
进剪辑前，每个生成 Clip 必须先过**闸 10**：用共享工具 `{TOOLS_ROOT}/video-frame-audit/frame_audit.py` 抽全帧 + 差分路由 + 分镜锚定取证，再逐帧视觉判定（含对抗性独立盲审），核对人物朝向 / 站位 / 道具 / 跨镜一致 / 身份 / 时长比例。完整规程见 `phase9-frame-audit.md`。
- 工具到 `AUDIT_READY`（证据齐 ≠ PASS）后才进入视觉判定；`AUDIT_INCOMPLETE` 硬阻断。
- 确定性时长/比例 `RISK_OR_FAIL_REVIEW` → 按 `FAIL_BLOCKED` 处理（除非显式接受测试风险）。
- **任一 Clip 未过（`FAIL_BLOCKED` / `AUDIT_INCOMPLETE`）→ 不进 Step1，先返修。**

### Step1 配乐意愿分层收集（分两步问，不一步问完）
**第 1 问：是否需要配乐**（单选，默认「不需要配乐」）
- 选「不需要配乐」→ 直接进入剪辑交接，不传任何配乐参数；
- 选「需要配乐」→ 继续问第 2 步详细偏好。

**第 2 问：配乐详细偏好**（仅选需要配乐时）
| 字段 | 说明 |
|---|---|
| 视频主题 | 如古风故事、美妆带货、旅行 vlog |
| 视觉风格 | 如暖调温馨、赛博朋克、清新自然 |
| 节奏偏好 | 如舒缓平稳、轻快活泼、激昂紧张 |
| 情绪氛围 | 如治愈、热血、悬疑、喜庆 |
| 分镜卡点描述 | 如「开头慢节奏 → 中段冲突加快 → 结尾舒缓」 |
| 用户额外偏好（可选） | 如「喜欢钢琴曲、要中国风元素、不要有人声」 |

### Step1.5 bgm-search：把配乐偏好转成检索关键词（原生子技能）
选「需要配乐」时，把表单 2 收集到的偏好组成 JSON，推断整体风格，生成 **3~5 个**配乐检索关键词（写进剪辑交接单，便于在目标平台/剪映音乐库检索）。
**输入 JSON**：`{theme 主题, visual_style 画风, rhythm_and_mood:{rhythm 节奏, mood 情绪}, script 文案, user_music_preference 用户配乐要求}`（字段可缺省/组合出现）。
**画风 → 乐器/曲风映射**：
- 国风水墨/古风 → 中国传统乐器（古琴、笛箫、琵琶）
- 赛博朋克/科幻 → 电子、合成器、未来感
- 日系治愈 → 轻音乐、钢琴、温馨柔和
- 写实电影 → 史诗配乐、交响乐、电影原声
- 卡通/2.5D → 轻快活泼、童趣
- 高端质感 → 简约大气、钢琴、弦乐
**生成规则**：
- **用户指定具体歌曲**（含《》或明确歌名/歌手，如「《小幸运》」「周杰伦 晴天」）→ Query **只返回歌名或歌手+歌名**，不加任何情绪/风格词。**不预判或拦截用户的配乐请求、不改写用户输入**——版权问题由剪辑工具侧处理，直接把歌名/歌手名写进用户偏好字段。
- **用户给风格描述**（如「爆款带货风格」「治愈系轻音乐」）→ 综合主题/画风/节奏情绪生成 3~5 个关键词，覆盖 情绪氛围 / 曲风乐器 / 节奏特征 / 主题意象 维度。
- 用户配乐要求优先体现；关键词易检索、与调性一致。
**合理性检查**（避免逻辑冲突）：美食不配伤感、古风武侠不配电子舞曲、高端奢品不配土嗨、儿童故事不配恐怖、励志运动不配慵懒慢节奏。
**输出**：`分析：<150~200字综合分析>` + `Query：关键词1 关键词2 关键词3 关键词4`（Query ≤15 字）。
> 示例：古风武侠题材、用户要「开头古琴/中段鼓点/结尾箫笛」→ `Query：古风 武侠 古琴 大气磅礴 侠义`；高端护肤广告 → `Query：高端 优雅 钢琴 轻奢 舒缓`；带货推广「爆款带货风格」→ `Query：抖音热门 带货 轻快 时尚 节奏感`。

### Step2 产出剪辑交接单
- **视频比例**：严格用 Phase6 确认的全局比例，不得更改；
- **片段顺序**：严格按 Clip 表顺序列出所有片段（Clip1→Clip2→…），顺序不能错；
- **重叠缓冲处理**（Clip 表有「重叠缓冲」标注时必填，无标注则不写本项）：逐条列出带缓冲的边界（Clip N → Clip N+1），说明后段开头约 1s 与前段结尾重复——剪辑时**在重叠区间内选动作最顺的切点、裁掉重复帧**，不固定从后段第一帧硬接；成片总时长以裁后为准；
- **配乐参数**：需要配乐时把偏好组装成清晰参数，严格按用户描述，不私自调整；
- **输出命名**：建议成片名 `最终成片_{项目名}_{时间戳}`。

剪辑交接单示例：
```markdown
# {项目名} 剪辑交接单
**全局比例**：16:9（不得更改）
**拼接顺序**：Clip1 → Clip2 → Clip3 → …（严格按故事板）
**重叠缓冲处理**：Clip2 开头约1s复现 Clip1 结尾（顾承泽抬手挡质问），在重叠区间内选最顺切点、裁掉重复帧；其余边界无缓冲
**配乐**：需要 / 不需要
  - （需要时）主题：古风故事；风格：暖调温馨；节奏：舒缓→中段加快→结尾舒缓；情绪：治愈；额外：要古筝、无人声
  - 检索关键词（bgm-search 产出）：古风 治愈 古筝 舒缓 温馨
**调色/卡点建议**：统一暖调；冲突段卡鼓点；结尾留 1s 黑幕尾韵
**成片命名**：最终成片_{项目名}_{时间戳}
```

### Step3 成片交付与验收
- 用户要求剪辑调整（顺序、配乐、节奏、调色）→ 仅重新剪辑，不重生成片段；
- 用户要求改某段 Clip 内容 → 回退 Phase8 重生成对应 Clip 再重剪；
- 用户满意 → 交付完成。

## 核心规则红线
- ❌ 禁止 Clip 拼接顺序与分镜表不一致，破坏叙事逻辑；
- ❌ 禁止用户明确不需要配乐时私自加配乐；
- ❌ 禁止用户未要求时私自加字幕、转场特效等额外内容；
- ❌ 禁止剪辑后更改视频比例，必须与全局比例统一。

## 退出条件
用户确认成片满意 → 全流程交付完成。
