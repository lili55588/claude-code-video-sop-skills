# 动作与肢体写法手册（skill 收录版）— 解决 AI 人物「动作僵硬 / 滑行 / 纸片人 / 站桩 / 动作乱」

> **上游源**：《【提示词创作第五十四节】让AI视频人物动作更自然的提示词写法》PDF +《AI视频人物动作提示词通用模板资料》DOCX +《AI角色调教三密码（为什么你的角色像NPC）》教程（刺猬星球，2026-06-11 收录）。上游出新版时重新同步本文件。
> **定位**：横切写法规则**三件套之三**——`micro-expression.md` 管**脸和眼睛**（表情/视线）、`voice-control.md` 管**嘴和耳朵**（音色/语气）、本手册管**身体和动作**（步态/联动/动作链/肢体小动作/多人互动）。
> **总纲**：视频和图片最大的区别是**时间变化**——人物要从一个状态过渡到另一个状态。不要只写"人物做了什么"，要写清"人物**怎样一步一步完成**这个动作"；动作方式必须**匹配角色身份与场景气质**（贵妃不能走出现代人的步子）。

## 一、核心配比：1 主动作 + 1 辅助动态 + 1 情绪状态 + 1 结束画面 + 1 镜头描述

- 只写一个小动作（抬眼/手指收紧/微微转头）→ 人物动得太少，**站桩**；动作堆太多（走路、转身、抬手、低头、回望、后退…）→ AI 抓不住重点，**动作乱**。每个分镜稳定的配比就是 1+1+1+1+1。
- **必须有结束画面/定格**：抬眼定格、对视停住、站定回望、动作完成后保持防备——"有主动作、有状态、有结束画面，人物才动得自然，也更有故事感"。
- 情绪状态的写法见 `micro-expression.md`（本手册不重复）；镜头描述在 video-sop 里必须落成**五段式镜头字段**（不能只写"镜头缓慢推进"）。

## 二、步态系统（"不要只写人物在走，要写清楚人物怎么走"）

图生视频第一步不是急着让人物动起来，而是先判断：**这个人物应该怎么走**（匹配身份+场景气质；古风项目对照 `guofeng-makeup.md` 角色方向库）。

| 步态 | 写法要点 | 适合 |
|---|---|---|
| 端庄步态 | 步伐端庄沉稳，速度较慢，身体平稳，动作安静克制 | 宫廷女子、贵族、正式场合 |
| 谨慎步态 | 步伐放轻，动作克制，速度较慢，像在小心观察周围，安静而警觉 | 夜晚古街、发现异常、危险场景 |
| 压迫感步态 | 目光锁定前方，短暂停顿后前行，每步很稳，重心平稳前移，肩膀略微下压 | 反派、高手登场、逼近目标 |
| 疲惫步态 | 步伐沉重，肩膀微垂，动作比平时更慢，带着疲惫压抑 | 受伤、失落、逃亡后 |
| 轻快步态 | 脚步有节奏，身体随步伐轻微起伏，衣袖发丝自然摆动 | 少女、轻松明亮场景 |

**通用模板**：`【人物】从【起始位置】向【方向】走去。步伐【步态特点】，速度【快慢】，身体状态【平稳/轻盈/沉重/紧张/克制】，整体动作符合【人物身份或场景气质】。`

## 三、身体联动 + 衣物发丝延迟摆动（消除"纸片人/提线木偶"）

**真实的人在动的时候，不会只有一个部位在动。**不要只写一个动作，要顺手补上其他部位的细微配合：

| 主动作 | 联动写法 |
|---|---|
| 抬头 | 视线慢慢上移，肩膀产生轻微变化，上半身稳定但有自然带动 |
| 转头 | **眼神先轻微偏移，头部再缓缓转向**（眼球先动、头后动，与 micro-expression 一致），肩膀和上半身小幅跟转 |
| 抬手 | 手腕先动，手指随后展开，肩膀轻轻带动手臂上抬，衣袖随动作下垂摆动 |
| 抱物侧转 | 手臂随身体微调位置，手指下意识收紧，衣袖出现细小褶皱变化 |

- **次级运动（物理惯性）**：衣袖、发丝、耳坠、衣摆随动作轻轻晃动；**"停步时衣摆有轻微延迟晃动"**——延迟摆动是真实感的关键。
- **重力感/体重感**：写清重心变化（重心微微向后偏移 / 重心慢慢上移 / 身体一侧充满重量感地靠在门框上），告别轻飘飘的纸片人。
- **辅助动态短句库**（每镜选 1 个，不要堆）：衣袖随动作轻轻摆动；发丝随移动轻微晃动；耳坠和发饰细小晃动；手指轻轻收紧又慢慢放松；眼神从前方短暂扫向一侧；经过柱子时身体轻轻侧过；抬手整理袖口动作克制；停步时衣摆轻微延迟晃动。

## 四、动作链公式（复杂/发力动作必用）

直接写动作结果（"少年挥动长杆"）太短——AI 不知道他怎么开始、怎么蓄力、怎么发力、停在哪里。

**完整动作链**：起始状态 + 触发原因 + 蓄力动作 + 身体发力（部位）+ 主动作（方向+运动轨迹）+ 情绪变化 + 结束定格 + 镜头配合

**挥杆案例拆解**：起始（双手握杆站定，目光锁定前方）→ 蓄力（手指收紧，手腕下压，肩膀绷住，重心降低）→ 发力（腰部和肩膀先带动，双臂跟着发力）→ 方向轨迹（长杆从身体右侧向左侧横扫，杆身划出快速弧线掠过镜头前方）→ 结束定格（身体略微侧转，长杆停在画面左侧，眼神依旧锁定前方）→ 镜头配合（轻微后退跟随横扫方向）。

- **简化链**（简单动作）：起始状态 → 主动作 → 辅助动作 → 结束状态。动态强可加到三步动作，**不要超过三步**。
- **与拆镜规则的边界**：动作链是**单镜内连续动作的过程写法**（连续动作无切镜禁拆）；遇到核心动作明显断点（抬手举杯→举杯喝下→放下杯子）仍按 Phase5.1 拆镜规则拆成多镜。
- **动作链模板库**（速取）：
  - 回头：原本面向前方→听到身后声响短暂停顿→头部缓缓转向，肩膀上身小幅跟转→视线落在声源，表情从平静变成警觉/疑惑/冷淡。
  - 起身：原本坐着→听到声音轻轻抬眼→手指按住桌沿，身体前倾，重心慢慢上移→缓缓站起，衣袖随动作落下→目光看向前方定格。
  - 靠近：目光锁定目标→短暂停顿→缓慢靠近，步伐稳定，肩膀微微下压→停在更近位置，眼神保持压迫/冷静/试探。
  - 后退：看到前方情况眼神微变→短暂停顿→重心向后移动，脚步缓缓后撤半步→重新站稳仍看向前方，带防备/不安。

## 五、肢体小动作词库（情绪的身体面）

人在情绪波动时会产生**自我安抚行为**——这是打破"模特站姿/摆拍感"的核心素材。四类按情绪选用：

**① 局促、紧张与防备**（核心=收缩身体+自我安抚）：重心局促地在两脚间来回切换；肩膀微微内扣、双臂向内收；手指抓弄/扣弄衣摆；在腰前隐蔽地绞着双手；双臂松散交叉、一只手紧抓另一侧手肘；大拇指指腹反复摩挲食指侧面。

**② 压抑、愤怒与隐忍**（核心=肌肉极度紧绷 vs 微小爆发力）：胸腔随急促不匀的呼吸剧烈起伏；双手在口袋里死死攥拳；死死抓住桌沿指关节泛白；用微微颤抖的手指缓慢整理领口（用日常动作掩饰天崩地裂，极强戏剧感）；指尖隐蔽地掐着掌心。

**③ 释怀、失落与卸力感**（核心=**重力感 Gravity**，紧绷肌肉失去支撑的物理惯性）：肩膀在挫败/释怀中微小下垂（配合叹气直接封神）；身体一侧充满重量感地靠在门框上；双臂松散垂在身体两侧彻底放松；缓慢呼气、整个上半身姿态随之软化；头部微微下倾、下巴靠近锁骨。

**④ 生活化毛边小动作**（专破"摆拍感/机器味"）：尴尬地揉后脖颈；推一下眼镜 / 将一缕碎发挽到耳后；一根手指有节奏地敲击杯缘或桌面；用大拇指轻柔摩挲木碗的纹理（触觉连接物理环境）；带着困惑轻挠侧脸。

**强目的性物理动词（硬规则）**：禁用"她的手在动 / 四处看 / moving hands"这类**无意义指令**——会触发 AI 随机"待机动画"；换成有生活目的的动作（摩挲花瓣、端茶吹气、整理花茎、收紧手指）。动作有目的，故事感才落地。

## 六、环境微介质与声画互证（连接人物与环境）

- **造介质**：水蒸气、晨雾、丁达尔光束、浮尘、飘落的叶片——微观物理介质是人物与环境的桥梁，瞬间消除"绿幕抠图感"。
- **环境微动叙事**：火光律动、阳光缓慢推移、热气升腾、风吹草动拂过碎发——环境要"叙事"不要"壁纸"，让空间跟着角色一起呼吸；禁用"狂风/宁静背景"这类极端或空洞词。
- **声画互证（目标平台适配）**：教程原版前提是"AI 视频只出画面、声音后期铺"，对目标平台**已过时**（8-A 本来就写环境音）。升级写法：写了视觉线索就**同步写对应环境音**——画面有"微风拂动野花和碎发"，环境音就写"草叶沙沙声"；画面有"壁炉火光律动"，环境音就写"柴火轻微噼啪声"。声画互证让目标平台的音频生成更稳（环境音写法见 `phase8-video-prompt.md` 可选增强）。

## 七、生成稳定性三铁律 + 守卫句（P8 用）

1. **微动态镜头优先固定镜头**：微表情/肢体小动作为主信息点的演技镜头，优先"固定镜头"或限定景别（中近景），防大动态运镜导致主体变形穿模。**范围限定**：仅演技镜头；叙事推进镜头照常按 Phase5 运镜交替规则，不许全片固定。
2. **减速副词压幅度**：在动作前高频使用 缓缓/轻轻/微微/极其微小地，压制 AI 的运动幅度，让情绪在慢节奏中发酵。**限场景**：情绪戏适用；营销快节奏、动作戏不滥用。
3. **动作指令不打架**：物理逻辑自洽——写了"双手端着茶碗"就不能再写"用手整理头发"；写了"垂眸深思"就不要写"直视镜头微笑"（与 Phase5.4 Step3 生理冲突同源）。

**守卫句（按镜头风险选用，不当万能尾缀）**：
- **防站桩**（人物动作少/独处/易不动的镜头加一句）：`人物需要完成明确的连续动作，不要只站立不动，不要只停在起势或微表情。`
- **防乱**（动作密集/易加戏的镜头加一句）：`动作保持简洁清晰，只完成这一组动作，不要加入无关动作、不要频繁切换姿势。`

**镜头搭配映射**（选五段式里的"运镜方式"时参考）：人物向前走→正面缓慢后退跟拍 或 侧面平移跟拍；情绪变化→缓慢推进；人物离开→背后跟拍；对峙→中近景轻微推进；独处→固定镜头；气场强→轻微环绕到侧前方。

## 八、多人互动回合制（多人画面分角色写）

只写"一男一女在房间里对视"→ 动态太少，视频里两人只会眨眼。**多人画面要分角色写，且每个人都有动作**：

**公式**：
> 主动方：位置 + 主动动作 + 情绪状态
> 回应方：位置 + 回应动作 + 情绪状态
> 互动关系：谁靠近、谁后撤、谁阻止、谁回应
> 结束状态：两人最后停在什么关系里（无声对峙/一人想走一人挽留/一人保护一人）

**动作回合示例**：男子靠近 → 女子后撤 → 男子抬手按住她手腕 → 女子抬头回应 → 两人对视停住。

**四个互动模板**（速取）：
- **对峙**：A 缓缓俯身靠近，目光压低带压迫感 → B 身体轻轻后撤，随后抬头看向 A，眼神紧张却不躲开 → A 抬手按住 B 手腕，B 手指微微收紧想抽回又停住 → 近距离对视，停在无声对峙。
- **告别**：A 站在门口准备转身离开 → B 短暂停顿后抬手轻轻拉住 A 衣袖 → A 脚步停住却没有回头 → 停在一人想走、一人挽留。
- **保护**：A 站在 B 身前目光警觉 → 前方有人靠近时 A 向前迈半步、抬手挡在 B 面前 → B 下意识后退半步神情紧张 → A 保持防备姿态，形成保护关系。
- **争执**：A 向前靠近像在逼问 → B 先低头沉默再慢慢抬眼回应 → A 抬手压住桌面身体前倾，B 手指收紧身体微微后撤 → 停在紧张克制的对峙。

**与姊妹手册联动**：多人对话的视线按 `micro-expression.md` 视线锚点路由（说话者看具名听者）；台词音色按 `voice-control.md` 各自独立设定；video-sop 正文仍**禁代称**——模板里的"角色A/B"落地必须写全名。

## 九、动作不自然判责树（成片后用户报"动作不对"）

| 现象 | 定位 |
|---|---|
| 滑行/腿部不协调/上半身僵直不跟动 | Prompt 只写了"做什么"没写"怎么做" → **修 Phase8** 补步态+身体联动；已写清 → 重抽该 Clip |
| 站桩不动/只眨眼/停在起势 | 缺主动作或动作太小 → **修 Phase8** 明确主动作+加防站桩句；Clip 表本身就没安排动作 → **回 Phase5** |
| 动作乱/频繁切换姿势/加戏 | 动作堆太多 → 按 1+1 配比精简，加防乱句 |
| 纸片人/轻飘飘没重量 | 缺身体联动/重心变化/衣物延迟摆动 → **修 Phase8** |
| 动作与身份气质不符（贵妃走出现代步） | 步态没写 → **修 Phase8**；上游就没定义气质 → 回 Phase4/P5 补 |
| 多人只会站着对视 | 没按回合制写互动 → **修 Phase8**；Clip 表没安排互动 → **回 Phase5** |

> 与 `rules-and-fallback.md` 回退表 C 联动。Phase8 属无损翻译——补联动/步态/守卫句可以，**新增 Clip 表不支持的动作必须回 Phase5**（与微表情改写边界同一条铁律）。

## 写完自查 6 问（P5.1/P8 动作镜头过一遍）

1. 写清"怎么动"了吗？（步态/速度/幅度匹配身份气质，不是只写"走/转身/抬手"）
2. 有身体联动吗？（主动作发生时肩膀/上身/手臂/头部有自然配合，衣物发丝有延迟摆动）
3. 配比对吗？（1 主动作+1 辅助动态，不站桩也不堆动作）
4. 有结束定格吗？（最后停在哪个姿势/眼神/关系里）
5. 动作有目的吗？（无"手在动/四处看"式空指令）
6. 多人分角色了吗？（谁先动、谁回应、停在什么关系，全名无代称）
