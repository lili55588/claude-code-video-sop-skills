# Phase 1 创意发散 + Phase 2 内容大纲

---

# Phase 1：创意发散（story-idea / ecom-idea）

## 核心定位
视频创作的**前置风险控制环节**。在用户需求模糊、方向不明确时，先共创思路、收敛创作核心要素（题材/风格/世界观/营销卖点），避免过早进入制作导致方向走偏、反复返工。

## 必须停留在本阶段的信号（命中任一即停留）
1. 用户在找方向/题材/风格抓手（「帮我想想拍什么」「给我几个创意方向」）；
2. 需求停留在题材/人设/卖点/受众/风格的讨论层，没有明确落地主轴；
3. 存在多个完全不同的可行方向，没有明确偏好；
4. 用户明确说「还没想好拍什么」「先聊聊思路」。

## 核心操作原则
1. **方向优先于制作**：用户还在想点子/求方向/对比方案时，不做任何制作类操作。
2. **提供可比较选项**：默认输出 **2~4 个差异化创意方向**，每个标注核心特点、适配风格、优势亮点，方便决策。
3. **按需收敛或延展**：
   - 主轴明确但缺落地方案 → 帮用户具体化、可落地化；
   - 想法很模糊 → 先做创意延展、给更多灵感，再逐步收敛；
   - 方向已明确且开始问制作参数/步骤 → 推进到 Phase2。
4. **编剧技能借用（已安装 `screenwriting-master` 时，可选）**：发散卡住或概念不够锐利时，可借用其**选题工具库**（主题/人物/空间/关系/标签碰撞五路径，内部引导不外露菜单）与**概念组合五法**（概念嫁接/时空错置/隐喻具象化/反转并置/规则反转）、What-if / How-to-Tell 双路径来锐化概念；产出仍按本阶段「2~4 个差异化方向」格式输出。用户要的是**纯剧本不出视频** → 整体交给 screenwriting-master，本技能不抢戏。

## 风格参考反推（上传风格图时的前置触发，源自 Super-I 2859）

**触发条件**：用户上传一张或多张图片 + 表达「按这个风格创作 / 复刻这种感觉 / 要这种调性」类意图时，**先启动五维反推把风格解析清楚，再继续创意**——不要拿着没解析的风格图直接往下走，否则后面 Phase1-5 都在没锁定视觉风格的情况下推进。

**默认定位 = 纯风格参考**：上传图默认只提取「画风 / 视觉规则」，**不绑定为任何具体场景的场景图**；仅当用户明确说「这张就是开场/某场景的实景」时，才另作场景参考交给 Phase7。

**五维反推框架**（把「好看」拆成可复用画面规则；参考图是成片截图/剧照时必须做全）：
① **镜头参数**：焦段、景深深浅、机位高度、构图方式、透视关系；
② **布光**：主光/补光/轮廓光/环境光的方向、强弱、软硬，自然光还是人工光；
③ **人物与场景**：妆造发型、服装质感、姿势、背景环境与天气；
④ **色彩与后期**：色彩风格、冷暖关系、对比度、肤色处理、颗粒/锐化/暗角等后期痕迹；
⑤ **复刻方案**：前四维收敛成一段可执行的视觉规则。

**产出「风格卡」（固定模板，字段口径写死、防字段漂移；随创意方向一并记录，作为全局视觉锚点贯穿全程，后续 Phase7/8 复用、不再重复反推）**：

```markdown
## {项目名}_风格卡
- **风格名**：（只收敛「画风/艺术形式」，如「日式赛璐珞动画风格」——禁把色调/光影/情绪/氛围词写进风格名）
- **全片视觉规则**：（一句话，把五维收敛成「这条片子整体长什么样」，供 Phase8 全片一致性约束 / {风格} 处直接引用）
- **五维明细**：①镜头参数… ②布光… ③人物与场景… ④色彩与后期… ⑤复刻方案…（留给执行层）
- **用户确认状态**：待确认 / 已确认 / 已要求重做（三选一；**只有「已确认」才允许 Phase7 A 分支复用**）
- **是否兼作场景参考**：否 / 是，指定为场景X / 是，指定为角色X / 是，指定为产品X（默认「否」=纯风格参考；选「是」**必须写明指定的是哪一类的哪一个**，Phase4 只能按此登记、不得被其他类吃掉）
```

> 五个字段缺一不可；`用户确认状态` 随用户反馈实时更新（用户说"就这个画风"才改成「已确认」）；`是否兼作场景参考` **必须由用户明确指定、AI 不替用户猜「是」**——默认「否」。这两个字段是 Phase7 风格闸与 Phase4 防误登记的判定依据（见 `self-check-gates.md` 闸 2c、`phase4-assets.md` 核心原则 5）。

**分层应用（关键，别一锅端）**：风格卡早做，但五维**分层喂给不同阶段**——
- **创作层（Phase1-3）**：只用「风格名 / 题材气质」这一层影响创意方向、大纲、剧本的调性与场景选择；镜头/布光/色彩参数此时**不介入**（过早灌入会干扰故事创作）；
- **执行层（Phase7 生图 / Phase8 视频）**：才用完整五维。Phase7 直接复用本风格卡的风格名+画面规则，不重复反推；Phase8 一致性约束的 {风格} 处取「全片视觉规则」。

**确认**：风格卡产出后请用户确认「这个画风方向对吗」，确认后锁为全局基准；用户要换风格则在此重做风格卡，下游随之更新。多张参考图风格冲突时，提示用户选主风格或指明各图用途（一张定画风、一张定光线等），不强行揉合成四不像。

**与 Phase7 的关系（消除"双入口"歧义，务必照此理解）**：全流程里**风格只确立一次**，本触发器只是把它**提前**到创意阶段。判断逻辑是单一决策、不是两个并行入口：
- **没上传风格图 → 本触发器不启动**，Phase1-2 正常走创意发散，**风格留到 Phase7 步骤二**按其"四种来源"流程首次确立（届时才反推）；
- **上传了风格图 → 这里产出风格卡**，Phase7 步骤二先判「有没有风格卡」：**有则直接复用、不再反推**。
即：**有风格卡＝Phase7 复用，无风格卡＝Phase7 才反推**，是同一决策的"提前/兜底"两条路径，任何一次创作只会命中其中一条。

## 故事类创意推演五步法（story-idea）
1. **锚定创意起点（找抓手）**：输入想法/主题→抓情感（设定极端角色/情境放大核心命题）；输入单张图片→抓画外音（推演画面发生的前后）；输入动态视频→抓冲突（明确核心矛盾点与角色目标）；输入具体场景→抓叙事位置（界定是开场、转折还是高潮结局）。
2. **注入脑洞元素（造反差，强行引入至少一种）**：**反常设定**（打破现实物理/社会规则，如买卖记忆）；**错位冲突**（极度不搭的元素强行组合，如社恐做外星谈判专家）；**概念翻转**（颠覆经典题材套路，如吸血鬼成了弱势群体）；**规则怪谈**（诡异世界法则，如说谎的人身上会长花）。
3. **提炼故事主轴（一句话公式）**：一个【什么样的主角】，在【什么情境下】，想要【达成什么目标】，但面临【什么核心阻碍】，最终【获得了怎样的改变/结局】。
4. **设计黄金开场（前 10 秒至少一种钩子）**：**视觉奇观**（不可能的画面，如雨滴悬停半空）；**信息差**（观众与角色认知错位，如对空气告白、拉远景发现他在悬崖边）；**情绪断崖**（极不合时宜的情绪爆发，如葬礼上狂笑）；**时间压迫**（开局即倒数，如距毁灭仅剩 3 分钟）。
5. **确定形态与类型（做包装）**：故事短片（强剧情/三幕式）/ 视觉短片（重情绪/意识流）/ 概念短片（高概念/极简风）；悬疑（控信息差/造未知）、情感（放大人性脆弱/两难抉择）、喜剧（极致错位/反差搭配）。

## 营销类创意方法（ecom-idea）
核心目标链：**抓注意 → 建立需求/信任 → 讲清卖点 → 促成行动（CTA）**。
- **钩子策略**（开场 3 秒）：感官冲击 / 反常识 / 痛点共鸣 / 悬念提问 / 强力对比。
- **卖点呈现**：过程演示 / 效果对比 / 细节特写 / 夸张反应。
- **感官卖点具象化四技巧**：特写载体（纹理/蒸汽/拉丝/碎屑/液体流动/反光层次）+ 动作触发（咬/掰/挤/撕/倒/按）+ 声音放大（咔嚓/嘶/噗/滴答）+ 夸张反应（旁观者/动物/物体/环境被震到），最后产品定格唤起想拥有。
- **反常识钩子**：先给出不可能、反常、超预期的结果，再解释「为什么能做到」。**翻车救场**：尴尬/崩溃临界点 → 产品一出手立刻封神。
- **平台适配**：抖音——前 3 秒钩子决定生死，强节奏强画面；淘宝——功能优先、细节特写、信任感>创意；小红书——真实种草感，开箱+体验流程；B站——信息密度高、逻辑清晰、趣味性>促销；朋友圈——品质感与信任、品牌早露出。

### ⚠️ 营销信息保真（合规红线，贯穿 Phase1~9）
- 用户提供的信息 = **事实基线**，不得更改或杜撰；缺失信息列「待确认项」提问，不自行补全。
- **禁止绝对化用语**（最/第一/100%）。
- **价格 / 促销 / 认证 / 销量数据：用户未提供 = 一律不出现**。
- 用户上传产品图时，提取：产品外观关键特征（形状/颜色/材质/结构）、品牌/Logo 位置与形态、场景风格与光影类型、图中所有文字逐行列出。

## 分场景对话策略
| 用户状态 | 应对 |
|---|---|
| 思路清晰、想法明确 | 补落地性建议，轻牵引到下一步，确认是否推进制作 |
| 有模糊想法、表达不清 | 先基于核心诉求延展 3~4 个具体方向，再引导选偏好 |
| 完全没思路、想要灵感 | 允许多轮发散，主动问有无喜欢的参考案例/风格，给不同赛道热门方向 |
| 连续不满意、卡住 | 切辅导模式，引导用户提供参考素材（风格图/对标视频），基于参考再调方向 |

## 外部信息增强
- **需要搜索**：实时热点、近期趋势、行业数据、具体产品参数、地域/历史文化设定。
- **无需搜索**：纯虚构故事、用户已给完整设定。
（Claude Code / Codex 有联网搜索能力时再用；没有就基于已知信息发散，并提示用户可补充事实。）

## 退出条件
用户明确选定某一创意方向、且同意继续推进制作 → 进 Phase2。

# Phase 2：内容大纲（故事大纲 / 营销创意大纲）

## 核心定位
复杂内容的**骨架搭建**，梳理底层逻辑，避免直接写剧本导致剧情崩坏/逻辑混乱/营销链路不通。**绝不是「缩写版剧本」**，只梳理脉络与因果，不涉及任何落地细节。

## 前置判定

### ✅ 可跳过（需**同时满足全部**）
- 总预估时长 ≤ 2 分钟；
- 单线叙事 / 单一营销逻辑，无并行线索、无时间跳跃、无多场景切换；
- 核心角色 / 产品 ≤ 2 个；
- 剧情转折 / 核心卖点 ≤ 2 个；
- 用户已提供完整内容梗概（含世界观、主线、核心设定）。

> **可跳过示例**：「做个 90 秒猫咪短视频，橘猫从沙发跳下打翻零食罐，委屈巴巴蹭主人求原谅，治愈风」→ <2 分钟、单线、1 角色、1 小转折 → 跳过，直接进 Phase3。

### ❌ 必须做大纲（满足任一）
- 时长 >2 分钟；多线叙事 / 时间空间跳转 / 人物关系复杂；核心角色或产品 ≥3；剧情转折 ≥3 或营销逻辑复杂（痛点→信任→转化多环节）；需求很模糊只有大方向。

> **必须做示例**：「古风穿越 3 分钟短片，现代织娘穿越古代靠织布逆袭，还有和皇子的感情线」→ >2 分钟、穿越设定、多人物、多转折 → 必须先做大纲。

## 操作步骤
1. **需求完整度校验**：已含完整脉络 → 告知可跳过，直接 Phase3；否则继续。
2. **生成大纲**：故事类按下方「故事大纲输出结构」，营销类按「营销大纲输出结构」。
3. **输出等确认**：呈现，按需迭代到满意，再进 Phase3。

## 故事大纲输出结构（story-idea，总字数 300~500 字）
1. **故事概览**：故事类型（情感/悬疑/喜剧/科幻等 + 该类型核心手法）；核心创意（用主轴公式 1~2 句概括：主角/目标愿望/阻碍冲突/主题）；节奏规划（快节奏适合喜剧悬疑 / 慢节奏适合情感文艺）。
2. **主要角色**：
   - 主角：姓名（暂定）/ 年龄身份 / 核心性格（2~3 关键词）/ **核心愿望**（最想要什么）/ **核心恐惧**（最害怕什么）/ **角色弧线**（开始时 vs 结束时的变化）；
   - 对手/阻碍角色：与主角关系、作用（制造冲突推动成长）、特点（不一定是坏人，但目标或价值观必须与主角对立）；
   - 次要角色（如有）：1~2 个 + 作用。
3. **故事结构**：三幕式框架（建置：日常世界+触发事件+核心问题 → 对抗：行动+冲突升级+中点转折 → 解决：高潮+结局+余韵）；按顺序列 **5~7 个关键情节点**，每点 1~2 句、只写因果不涉细节。
4. **主题与情感**：核心主题（用故事而非说教表达）；情感曲线（观众情绪从什么到什么、关键情感节点在哪）；共鸣点（观众的投射点）。

## 营销大纲输出结构（ecom-idea）
1. **广告类型**：产品硬广(强转化) / 产品软广(体验种草) / 品牌品宣(认知建立)；目标受众；情感共鸣（爽感/安心感/向往感/紧迫感）；钩子策略 + 卖点呈现方式。
2. **创意规划**：节奏规划（1~3 个核心场景）；创作原点（一个具象化核心卖点，如「一撕即净的爽快感」）；广告目标（CTA 必须是**单一行动指令**）；**卖点金字塔**——核心卖点占 70%（用最强烈的视听语言证明）+ 支撑卖点占 30%（1~2 个，每个 1~2 个镜头快速带过）；**5 个关键情节点**：①激发事件（3 秒钩子，强视觉/强反差）→ ②冲突/痛点/卖点 → ③解决方案（创意镜头/戏剧动作/感官特写展示产品）→ ④强化证据（细节拉满、反应夸张）→ ⑤行动收束（单一 CTA）。
3. **广告大纲**：基于创意规划写一段话、强创意、高吸引力的大纲。**一个卖点/创意贯穿全文**；开头 3 秒极度吸睛；抽象卖点用「特写+动作+放大音效+夸张反应」；结尾落 CTA、不写未确认的价格优惠；全程每句对应「看到什么/发生什么」，不写模糊口号。

## 严格禁止（踩坑红线，按分支区分）
**故事类大纲**：
- ❌ 禁止写任何台词、对话；
- ❌ 禁止写分镜、镜头运动、景别；
- ❌ 禁止写细节动作描写。
  - 错（越界）：`苏晚加班到凌晨3点揉了揉发酸的眼睛，伸手碰插座突然触电，眼前一黑晕过去，醒来在破茅草房…`
  - 对：`现代织娘苏晚加班时触电，穿越到古代成为织造局小丫鬟`

**营销类广告大纲**（边界不同，勿套用故事红线）：
- ✅ **允许且应当**用感官镜头语言证明卖点（特写/微距/跟拍 + 动作 + 放大音效 + 夸张反应——这是官方 ecom-idea 的硬要求）；✅ 允许口播 CTA 短句（如「夏天的第一口脆，就选梦龙」）。
- ❌ 仍禁止写成逐镜编号的分镜表、禁止标时长（那是 Phase5 的活）；❌ 禁止写完整多轮对话；❌ 禁止模糊口号（每句对应「看到什么/发生什么」）；
- ❌ 禁止绝对化用语、禁止杜撰未提供的价格/促销/认证/销量数据。

## 回退
用户要「再发散方向/换切口/改结构」→ 先回 Phase1 产多方案，选定新方向后再回本阶段调大纲，不在原大纲上反复硬改。
