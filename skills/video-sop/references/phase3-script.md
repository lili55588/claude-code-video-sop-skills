# Phase 3：剧本创作（story-script / ecom-script）

## 核心定位
所有制作环节的**核心依据**，产出可直接拆解成分镜的完整拍摄剧本。后续素材挖掘、分镜、视频生成都基于本阶段。
> 硬规则：长视频、多剧情线、所有营销广告脚本，**必须经过本阶段**，禁止从创意/大纲直接跳到分镜。

## 前置判定
### ✅ 可跳过（满足任一）
- 用户已给按场景组织的完整剧本（含每场动作描述、对话、场景说明）；或给了标准化分镜列表（含每镜画面、台词、时长）。
### ❌ 必须创作
不满足跳过条件的都必须生成完整剧本。

## 创作原则
1. **剧情优先**：服务核心剧情/卖点表达，不堆砌无效情节。
2. **视觉优先**：写可拍摄的画面，减少抽象心理/抒情描写（硬规则见下节）。
3. **逻辑自洽**：人物行为符合人设、前后一致，无无解释的剧情漏洞（硬规则见「剧情逻辑一致性」）。
4. **节奏把控**：张弛有度，匹配故事/广告类型。
5. **可拍摄性**：环境/道具/动作/对白符合时代、地域、身份设定，不出违和元素。

## 可拍摄性硬规则（只写镜头可验证的信息）
动作描述里只允许三类内容：**可见动作与表情、可闻声音与台词、可被当前场景直接验证的物理细节**。禁止直接写人物心理、主观判断、作者解释、抽象感受。

- **「内在认知」改写为「外在证据」**：不写「觉得、意识到、明白、想起、痛苦、害怕、被背叛」等脑内信息，改写为停顿、视线、动作、道具、环境变化、对白，或镜头能直接捕捉的证据。
- **跨时间比较先建画面锚点**：不直接写「和十年前的一模一样」；必须先在前文或当前场景提供可见依据（旧照片、旧物、伤痕、编号、缺角、录音、独立成场的闪回），再完成比较，且优先写具体可见细节、不写抽象结论。
- **不可拍信息要转译、不要硬删**：用户输入里本就含回忆/情绪/抽象概括/人物认知时，保留其叙事功能、改写为可拍表达——情绪→行为反应；判断→可见依据；抽象概括→具体细节；认知变化→触发过程与后续动作；回忆→痕迹或闪回（独立可拍场景 + 清晰触发物）。
- **用行为代替情绪标签**：「他很痛苦，觉得被背叛了」→「他盯着手机上的转账记录，半天没说话。杯子在手里越攥越紧，突然『啪』地砸进水槽。」
- **用具体细节代替抽象概念**：不写「熟悉感 / 命运的嘲讽 / 压抑的气氛」，改写伤口、污渍、物件摆放、空间距离、停顿、呼吸、噪音、灯光状态。
- **不给导演写死分镜（故事类）**：除非信息呈现依赖特殊镜头，否则不密集指定「特写/推镜/慢动作/蒙太奇」，只交代场面调度、关键信息、人物动作和节奏点，镜头选择留给 Phase5。**营销类例外**：产品重点展示按官方 ecom-script 要求使用 `【产品特写】` 标识，且特写描述要更细致——这不算"写死分镜"。
- **高频错误词**（出现时优先检查是否不可拍）：觉得、意识到、明白、想起、仿佛、似乎、痛苦、难过、愤怒、害怕、背叛、熟悉、命运、压抑、一模一样。不是绝对禁用，但只要表达的是角色脑内判断而非可见事实，就必须改写。
- **逐句自检四问**（生成剧本后执行）：①摄影机能拍到吗？②麦克风能录到吗？③演员能演出来吗？④这一句写的是「角色知道了什么」还是「观众看到了什么」？——前 3 问任一不成立、或第 4 问是前者，必须改写。

### 微表情钩子（轻量，详见 `references/micro-expression.md`）
- 情绪节点默认**微档**写法：用强度天花板词（淡淡的/几不可察的/克制的/一闪而过的）封顶，不写"大笑/嚎啕大哭/极度震惊"等裸奔大情绪词——AI 会把强情绪在整段时长里锁死在 100%，成"动态面具"。
- 关键情绪转折处**预埋一个物理 Transition**（闭眼、深呼吸、动作停顿、转头、放下手中物品），Phase5 拆镜要靠它切断前一个情绪。
- 镜头级微表情细节（眼睑/咬肌/喉结等部位写法）**不在剧本堆**，留给 Phase5.1 的速用卡处理。

### 声音钩子（轻量，详见 `references/voice-control.md`）
- **台词的标点就是节奏器**：逗号=短停顿、句号=完整停顿、省略号=拉长停顿带犹豫、感叹号=情绪爆发——写台词时按角色当下情绪**有意设计标点**，不随手乱点（标点计入 Phase5 计时字数，省略号自带停顿时长）；感叹号不堆砌（连续感叹号=全程嘶吼，同"大情绪词裸奔"）。
- 情绪递进的长对话按递进**分句写**（先缓后快/先抑后扬），给 Phase8 的逐句声音标注留好层次。
- 角色声音设定（年龄感/声线质感/语速/情绪底色/发声习惯）**不在剧本写**，登记在 Phase4 素材清单的音色描述里。

### 编剧增强钩子（轻量，已安装 `screenwriting-master` 时可选）
- 剧本初稿生成后、交用户确认前，可选用其**「剧本医生」**做一遍轻量诊断（节奏/逻辑漏洞/直白对话/潜台词），诊断意见仅作修订参考——产物仍守本阶段强制格式与可拍摄性硬规则，不改输出结构。
- 由其产出/协助的剧本若含**电影名/导演名/名人参照**（其剧本创作阶段的合法内部工具），落盘本阶段剧本前一律清洗为中性描述（下游 Prompt 红线，见 SKILL.md 跳步判定）。

## 剧情逻辑一致性（不仅拍得出来，还要讲得通；故事类为主）
任何关键事件都必须满足：观众能看见原因，角色的反应符合已知信息，结果符合故事设定。（营销类对应适用：产品使用过程与效果展示同样须符合物理常识，效果夸张要有产品机制/演示动作支撑，不得无铺垫"瞬间见效"。）
- **先判定世界规则**：现实主义/都市/古风/非超现实科幻 → 默认遵守现实物理、日常常识、社会规则、技术边界，禁止无铺垫的「按一下按钮物体消失 / 隔空获知信息 / 伤势瞬间恢复」。奇幻/超现实/高概念科幻 → 允许非常规事件，但必须**先建规则再按规则执行**：至少交代触发条件、作用范围、限制或代价中的两项，后文不得临时改规则。
- **关键结果前必须有可见成因**，不跳过中间必要步骤；所有「异常高效」的结果都要特别审查——补机制、工具、铺垫或代价，否则改回符合常理的表达。
- **角色行为符合其身份、目标、处境和刚获得的信息**；禁止为推进剧情让角色违背常识、职业能力与已知风险判断。
- **角色知道什么必须有来源**：亲眼所见 / 亲耳所闻 / 他人明确告知 / 文件设备记录 / 既有经验 / 清晰推理链，禁止说出本不该知道的信息。
- **时间标记前后一致**：「几分钟后 / 两小时后 / 次日 / 上周 / 当晚」一经出现，后文严格沿用，不得随意缩短、拉长或改口。
- **转折可以突然，但不能无因**：转折前至少一个前置信号——异常行为、可疑道具、反常对白、被忽略的细节、世界规则说明、倒计时、限制条件。

## 主观时空（闪回/梦境/想象/幻觉）
- 主客观场景变化必须**单独成场**：主观场景在地点后加后缀，如 `### 场景4：家里客厅（回忆），夜晚，内。`、`街道（幻觉）`。
- 进出主观时空用**叙事标记**：在 △ 动作行写（闪回进入）/（闪回结束）/（梦境进入/结束）/（想象进入/结束），并给出清晰触发（照片、旧物、声音、地点、动作）。
- 闪回段落必须写成**独立、可拍摄的场景**，服务当前情节（补充角色当下判断、关系变化或信息揭示）；段落结束后回到当前时空，让角色在当下有可见反应或行动承接；禁止把关键因果偷渡成一句作者解释。

## 对话/文案写作（按分支）
**故事类·对话写作**：
- 凡**信息揭示、关系变化、谈判对抗、误会澄清、约定/命令/请求**等需要语言推动的情节，必须写出有效对话，不能只写动作。
- 对话符合身份/年龄/性格；好的对话有潜台词；停顿与沉默也是对话的一部分；不直接说出主题、避免说教与刻意「金句」。

**营销类·文案写作**（勿套用故事对话规则）：
- **口语化**：像对人说话一样自然；**利益导向**：强调用户能获得什么、而非产品有什么；**简洁有力**：删多余词、每句有价值；**避免专业术语**：用目标受众能理解的语言。
- 文案功能：传递卖点 / 引发共鸣 / 建立信任（证据、证言）/ 促进行动（明确 CTA）；关键卖点在开头和结尾重复强化（记忆点设计）。

## 剧本核心逻辑（故事类）
- **三幕式结构**：第一幕建置（引入人物+建世界观+抛核心冲突）≈25% → 第二幕对抗（冲突升级+人物成长+情节推进）≈50% → 第三幕解决（高潮+冲突解决+结局）≈25%。变体：线性 / 倒叙插叙（悬疑）/ 多线交汇（复杂）。
- **人物弧光**：开场设初始状态（缺点/困境/执念）→ 冲突迫使改变 → 结尾完成转变、与初始呼应。
- **情绪曲线**：起伏交替（情绪低点后升级冲突、高点后短暂舒缓）；**核心情绪点（感动/震撼/反转）放结尾前 10%**；首尾情绪呼应。
- **伏笔与回收**：所有关键道具/信息/细节后续必有回收；埋设自然、回收时机"快忘没忘"时揭晓、重要伏笔多次铺垫。

## 营销脚本核心逻辑（营销类）
- **问题-解决方案结构** + **注意力曲线管理**（黄金 3 秒钩子）+ **记忆点设计**（slogan/记忆锚点）+ **说服逻辑链条**（痛点→信任→证明→转化）+ **品牌调性落地**。
- 营销带货类**先调「营销文案大师」能力**：先检索/归纳同类爆款话术结构，再写台词，保证转化率（本机无该子智能体时你自己承担这步）。

## 操作步骤
1. **剧本需求校验**：已有完整剧本 → 跳过进 Phase4。
2. **核心剧情对齐自检（输出前执行，不写进最终输出）**：逐条提取用户确认的核心剧情节点/人物设定/关键情节/结局要求 → 对照剧本逐项核对，每个要点都在对应场景完整呈现，**不得为追求流畅删改用户要求的关键内容**；开头/高潮/结尾三个核心节点必须完全符合用户要求，加创意只能叠加不能改核心。
3. **生成剧本**：故事类 story-script、营销类 ecom-script，按下方格式输出。
4. **保存并确认**：存为项目文件夹下的 `F:\claude长视频创作\{项目名}\{项目名}_剧本.md`（这是本项目第一份落盘文件，直接写到该完整路径即可自动建好项目文件夹；归档规则见 SKILL.md §3），展示，迭代到满意，再进 Phase4。

## 强制格式（原生 slug 剧本格式）
- 文件主标题 `# {项目名}剧本`（与文件名对齐）；`## 故事类型`（或营销的 `## 投放渠道`）、`## 人物清单`（营销为 `## 产品/角色清单`）、`## 分场剧本` 用 `##`。
- **人物清单**：每条一行 `角色名｜性别｜年龄｜身份｜核心性格特征｜与XX关系`。
- **场景标题（slug 行）**：`### 场景X：地点，时间，内/外。`
  - 地点 = "发生在哪里"（**只写主场所**，禁把"展柜区/走廊/门口/舞池边"等子区域写进标题；强调子区域用 △ 动作行标注，如 `△【展柜区】…`）；内/外 = "摄影机所处空间"；时间 = **故事时段**（白天/夜晚/清晨/黄昏——不写灯光或画面状态如"晚宴灯光/黑暗/应急红光"，灯光变化写进 △ 动作行）；主观场景在地点后加后缀（回忆）/（梦境）/（幻觉）。
  - **场景拆分**：仅当"时间"或"内/外"变化才新开场景；同一物理空间（不同叫法/包含关系仍算同一场景）小区域移动 / 人物增减出入场 / 画面逻辑突变（灯光骤灭/警报响起）**不拆场**，用一条 `△` 动作行衔接或作转折断点；主观时空（闪回/梦境）是不同场景、必须拆。
  - **外景天气**：室外场景或涉及从室内看室外的场景，**第一条 △ 动作行必须补天气**（晴/阴/雨/雪/雾）；纯室内不强制。
- **场景正文**：`**人物**：…` → `△` 动作行（只写可拍到的：环境+关键动作+关键道具/信息点，也可写停顿/欲言又止等可表演状态；禁抽象抒情/心理；**除台词外禁止比喻、成语**）→ `角色（动作/表情）：台词`。
  - **故事类**：旁白/内心独白作为说话人直接写（`角色内心独白：…`、`旁白：…`），不用方括号标记，不密集写镜头指令。
  - **营销类**（官方 ecom-script 标记）：旁白用 `【旁白】`（必须与当前画面一致、无音画违和）；产品重点展示用 `【产品特写】`（标了特写的镜头描述要更细致、并关注场景/产品/角色的上下文一致性）。
- 场景间 `---` 前后各留空行。预估总时长写在开头。

**故事类示例**：
```markdown
# 苏晚织梦剧本

## 故事类型
古风穿越 / 励志逆袭

## 人物清单
- 苏晚｜女｜24岁｜现代织娘穿越成织造局小丫鬟｜聪慧冷静、有现代技术
- 张嬷嬷｜女｜50岁｜织造局管事嬷嬷｜刻薄势利
- 李管事｜男｜40岁｜织造局管事｜识货、重利

## 分场剧本

### 场景1：织造局偏院机房，清晨，内。
**人物**：苏晚、张嬷嬷
△破旧机房，堆着坏织布机，晨光从窗棂斜照。苏晚蹲在老织布机前调试综丝，指尖把竹制零件卡进卡槽。
苏晚（自言自语，微微一笑）：古代的织布机改一改，也能织出现代的提花纹样。
△门口传来脚步，张嬷嬷抱着胳膊站在门口，脸色阴沉。
张嬷嬷（阴阳怪气）：哟，来了三天什么活都没干，在这拆织布机呢？

---

### 场景2：织造局管事房，正午，内。
**人物**：李管事、苏晚
△桌上摆着进贡布料样。李管事拿着苏晚织的提花布料翻看，手指摩挲纹样。
李管事（惊讶）：这真的是你织的？这种纹样我在宫里贡品里都没见过！
苏晚（点头，神色平静）：如果管事愿意给我机会，我还能织出更好的。
```

> 等价变体：也可用「`### 第X场 【场景名】` + `**场景**/**动作画面**/**台词**` 加粗字段」格式（早期主手册写法），信息等价；新建项目**默认用上面的原生 slug 格式**。营销类同结构，把 `## 故事类型`→`## 投放渠道`、`## 人物清单`→`## 产品/角色清单`，台词突出卖点与 CTA；产品重点展示用 `【产品特写】` 标识。
> **营销类音效同步规则**：产品特殊音效必须与画面动作同步——薯片**被咬碎时**才可加咔滋音效、可乐**被拧开时**才加噗呲音效，禁止音效脱离动作出现。

## 时长预估（写入剧本后按字数预估）
| 剧本字数 | 预估时长 | 说明 |
|---|---|---|
| 500 字 | ≈1 分钟 | 对话多≈1分钟；动作描写多可能仅几十秒 |
| 800 字 | 1.5~2 分钟 | 标准故事短片密度 |
| 1200 字 | 2.5~3 分钟 | 情节复杂/情感铺垫长 |
> 字数系数：对话为主 每秒约 8~9 字（每分钟≈500字）；动作/场景描写为主 每秒约 5~7 字（每分钟≈300~400字）。
