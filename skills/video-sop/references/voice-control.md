# 声音结构写法手册（skill 收录版）— 解决 AI 角色「开口就假」/ 语气平 / 千人一声

> **上游源**：《【提示词创作第四十八节】如何控制 AI 角色的语气、语速和音色？》（刺猬星球教程 PDF，2026-06-11 收录）。上游出新版时重新同步本文件。
> **定位**：横切**写法规则**（地位同 `micro-expression.md`）——那份管**脸和眼睛**（表情/视线），这份管**嘴和耳朵**（音色/语气/语速）。不是素材、不是独立 Phase；落点在 P4 音色登记与 P8 音色注入。
> **总纲**：画面和声音是两套控制逻辑。画面靠场景/光线/镜头/动作/景深/构图；声音同时携带**年龄感、声线质感、语速、停顿、气息、情绪底色、发声习惯、重音位置**。「愤怒地说」「悲伤地讲」只描述了情绪**结果**、对 AI 约束极弱——必须写**声音过程**：不告诉 AI「这个人很生气」，告诉 AI「**这个人是怎么把生气说出来的**」。控制 AI 声音，本质不是堆形容词，而是搭建**声音结构**。

## 本 skill 内怎么用（按 Phase 路由）

| 阶段 | 用哪几节 | 干什么 |
|---|---|---|
| Phase3 剧本（轻量） | 第三节 | 写台词时**有意设计标点**（标点=节奏器，且计入 Phase5 计时字数）；情绪递进的长对话按递进分句写。钩子见 `phase3-script.md` |
| Phase4 素材清单 | 第一节 | 有台词主要角色的音色描述在官方 6 字段后补 3 个**扩展字段**（声线质感/情绪底色/发声习惯）；多角色拉开听感差异 |
| Phase5 分镜 | 第三节 | 台词跨镜承载时**标点一字不丢**（节奏+计时双重依据）；计时基准不变（每秒 5 字含标点） |
| Phase8 Prompt（**主战场**） | 第二/四/五/六/七节 | 每句台词的音色 = **基础音色**（素材清单原样复用、跨任务一字不差）+ **本句声音过程**（重音/停顿/尾音/音量/气息）；多句台词逐分镜递进；禁裸情绪词 |
| Phase8 生成结果校验 | 第八节 | 成片声音不对时按判责树定位（重抽 / 修 P8 / 回 P4 / 回 P3） |

> 加载策略：有台词/旁白的项目，Phase8 写 Prompt 前至少过一遍**第七节速用卡**；对话戏重（谈判/对峙/情感戏）整读本文件。用户报「开口就假 / 语气很平 / 角色听起来都一样 / 语速不对」类问题时，无论项目类型直接加载本文件修复。

## 一、基础音色公式（P4 登记，P8 复用）

**公式：年龄感 + 性别/声音类型 + 声线质感 + 语速 + 情绪底色 + 发声习惯**

与素材清单官方 6 字段的对接（官方字段一个不动，公式拆成「官方 6 + 扩展 3」）：

| 公式成分 | 素材清单字段 | 示例 |
|---|---|---|
| 年龄感 | 官方 `年龄`（精确整数） | 年龄60 |
| 性别/声音类型 | 官方 `性别` | 性别男 |
| —— | 官方 `语调` / `音高` / `口音` | 语调低沉威严，音高偏低，口音无 |
| 语速 | 官方 `语速` | 语速缓慢 |
| **声线质感**（扩展1） | 追加在官方 6 字段后 | 声线质感低沉干哑、带阅历感 |
| **情绪底色**（扩展2） | 同上 | 情绪底色冷静、克制、审视 |
| **发声习惯**（扩展3） | 同上 | 发声习惯：句间自然停顿、咬字清楚、尾音略微下沉 |

**规则**：
1. **只写角色身份 ≠ 写声音**。「一个年迈谋士说话」「一个冷艳公主开口」只告诉 AI 角色是谁，AI 只能瞎猜声音。❌「一个年迈谋士严肃地说」→ ✅「60岁年迈谋士声音，声线低沉干哑，带阅历感和朝堂威压，整体语速缓慢、每句话之间有自然停顿，情绪底色是冷静、克制、威严，音量不高但咬字清楚、尾音略微下沉」。
2. 扩展字段写**可听参数**（干哑/清亮/醇厚/沙哑；停顿/咬字/尾音/气息），比喻（"像是在警告"）只能作补充、不能替代参数。
3. **多角色差异化**：官方要求 ≥4 字段明显不同，扩展字段同样参与拉差异——少年将军（压着怒气、语速偏快、尾音短促）/ 年迈谋士（低沉干哑、缓慢、字间停顿）/ 冷静女主（气息稳定、音量不高、字字清楚），绝不让所有角色听起来是同一个人。
4. **合规**：禁止写"声音像{名人名}"（与 8-A 禁名人名同源）。

**速用示例（直接当模板改）**：
- 清冷女主：`性别女，年龄25，语调克制隐忍，音高偏低，语速缓慢，口音无，声线质感柔和偏低带失落感，情绪底色克制与失望，发声习惯：气息稳定、后半句逐渐变轻、咬字清楚、句尾轻收`
- 反派军师：`性别男，年龄35，语调冷静带讽刺，音高偏低，语速缓慢，口音无，声线质感偏低略带沙哑，情绪底色掌控感、冷意、危险感，发声习惯：音量不高但压迫感强、重点词轻微加重、句间有停顿、尾音下沉`

## 二、情绪的声音结构（语气控制，P8 本句声音过程）

情绪词是结果，声音过程才可执行。把情绪拆成 **音量 / 语速 / 停顿 / 气息 / 重音 / 尾音** 的变化：

| 情绪 | 声音结构 |
|---|---|
| 愤怒 | 音量提高，语速加快，句尾收紧 |
| 压抑的怒（更高级） | 音量不高但压迫感强，重点词轻微加重，句间停顿，尾音下沉——**越平静越有威胁感** |
| 悲伤 | 音量下降，语速缓慢，气息变轻 |
| 强忍悲伤 | 气息稳定但后半句逐渐变轻，句尾轻收，不哭喊 |
| 冷静 | 语速稳定，语气平和，字字清晰 |
| 不安/期盼 | 语气带轻微颤抖，尾音拖长或轻微上扬 |
| 坚定/决断 | 语气加重，语速果断，尾音下沉无拖延 |

**错误 vs 正确（反派军师案例）**：
- ❌ `"你以为自己赢了？"他愤怒地说，"不！你只是走进了我的陷阱！"`（只有情绪结果，AI 不知如何发声）
- ✅ `反派军师音色，35岁男性，声线偏低、冷静、略带沙哑，语速缓慢，情绪底色是掌控感、冷意、讽刺感和危险感，音量不高但压迫感强，咬字清楚，重点词轻微加重，句间有停顿，尾音下沉，越平静越有威胁感。台词：「你以为自己赢了？不，你只是刚刚走进我为你准备好的局。」`（前半句平静带讽刺，后半句靠停顿和尾音下沉展现压迫）

**写法要点**：
1. 本句声音过程从 **重音落点 / 停顿方式 / 尾音处理 / 音量变化 / 气息状态** 里挑 **2~4 个最关键**的写，不全堆（堆满=没有重点）。
2. 重音标注引用台词片段时用 `'…'`，**不用「」**（「」专属台词本体，防止与 8-A 台词引号混淆）：`'从来'两个字加重，句尾轻轻收住`。
3. **声画同源**：声音颤抖时画面要有对应微表情（下巴颤动/嘴唇抿紧，见 `micro-expression.md`），防"声音在哭、画面在笑"的音画违和。

## 三、标点符号 = 节奏器（语速控制，P3 写台词时预埋）

| 标点 | 声音效果 | 用法 |
|---|---|---|
| 逗号 ， | 短暂停顿、轻微减速 | 制造层次感、矛盾感："我明白你有你的原因，但我还是不赞同。" |
| 句号 。 | 完整停顿、语速正常 | 决心与结束的信号："我们已经说清楚了。" |
| 省略号 …… | 拉长停顿、放慢、带犹豫/未尽之言 | "你……难道就这么离开了吗？" |
| 感叹号 ！ | 情绪爆发、语速加快、急促 | "你怎么能这样对我！"——**不堆砌**（连续感叹号=全程嘶吼，同"大情绪词裸奔"） |
| 短句 | 节奏加快 | 紧张/对抗段落 |
| 长句 | 情绪铺陈 | 倾诉/回忆段落 |

**本 skill 的三条对接规则**：
1. **标点计入计时**（Phase5.2：字数含标点 ÷5）——省略号自带停顿时长，节奏和计时天然一致；台词跨镜拆分时**标点一字不丢**。
2. PDF 里"换行=段落停顿"在 8-A 语境下不可用（台词在「」内单行）——等价手段是 **Phase5 的台词跨镜承载**：把长台词按情绪节点拆到连续多个分镜，就是即梦语境的"换行"。
3. 音色设定里的"语速偏慢/偏快"**不改变 ÷5 计时基准**（官方定值，见 `rules-and-fallback.md` 附录 D.3），只在 Phase5.2 整体节奏微调时酌情参考。

## 四、多句台词：按时间顺序写声音递进（P8 核心技法）

长对话的情绪是逐渐变化的，声音必须**随情绪递进**，不能一整段只给一个情绪。典型弧线：开始语速较慢带犹豫 → 情绪升温语速加快甚至爆发 → 最后平静或释然、语速再次放慢。

**模板**：
> 角色声音设定：【年龄感】+【声线质感】+【整体语速】+【情绪底色】
> 第一句：【声音状态】+【语速】+【停顿】
> 第二句：【情绪变化】+【重音】+【句尾】
> 第三句：【音量变化】+【爆发或收住】
> 第四句：【最终情绪落点】

**示例（克制→愤怒）**：
> 30岁左右的男性声音，声线低沉，语速先慢后快，情绪从克制逐渐转为愤怒。
> 第一句'你终于肯说实话了'，音量不高，语速缓慢，尾音下沉。
> 第二句'这些年，你一直在骗我'，语速加快，'一直'和'骗我'加重。
> 第三句'你到底把我当成什么'，音量明显提高，气息变重。
> 第四句'回答我'，短促有力，没有多余尾音，情绪彻底爆发。

**映射到 8-A 分镜段**（与微表情三段式时间段语法同构）：多句台词在 Phase5 已跨分镜承载——每个分镜的台词带**它自己那一段的声音过程**，分镜序列天然就是声音递进的时间轴：
> 分镜1：…沈知意@图片2说：「你终于肯说实话了。」音色：{基础音色}。这句话音量不高，语速缓慢，尾音下沉。
> 分镜2：…沈知意@图片2说：「这些年，你一直在骗我。」音色：{基础音色}。这句话语速加快，'一直'和'骗我'加重。

**跨 Clip 声音弧线**：对话跨多个 Clip 时——
1. 同一角色**基础音色一字不差**复用（各 Clip 独立并行生成，基础音色变了=换人声）；
2. 声音过程接着上一 Clip 的情绪落点继续递进，**不无过程回跳**（上一 Clip 末句已爆发，下一 Clip 不能直接回到平静——要么延续、要么写出可听的释放：长呼一口气后语速放缓）；
3. **情绪残留有声音面**（哭后第一句带轻微鼻音、吸气声；压怒后咬字偏重、句间停顿变长）——与闸 6 情绪残留四类联动。

## 五、多人对话：拉开声音差异 + 逐句标注

每个说话角色先给**独立声音设定**，再逐句标语气变化。差异维度：年龄层 / 音高 / 语速 / 质感 / 情绪底色（对应 P4 ≥4 字段差异）。禁止多角色共用一段音色描述。

**对比示例（皇宫对话）**：
> 皇后：50岁左右，声音低沉且充满威严，语速适中，情绪底色为坚定与不甘，语气不急不缓、每句话结束时有明显停顿，表现出对权威的掌控感。皇后：「太子，你已经知道这个决定无可更改。」
> 太子：30岁左右，声音清晰且略带焦虑，语速稍快，情绪底色为不安和无奈，语气时而急促、时而拖延。太子：「母亲，我明白，但这样的决定真的对国家有益吗？」

**逐句标注示例（父女对话，节选）**：
> 父亲：「你已长大，不能总是依赖我，未来的路要自己走。」——'你已长大'语气平稳、语速适中；'不能总是依赖我'语气逐渐加重、语速略微放慢；'未来的路要自己走'尾音下沉、语速慢且坚定。
> 女儿：「可是，父亲，我并不准备离开……」——'可是'语气较轻带犹豫；'我并不准备离开'语气渐加重、语速加快；省略号处语速放慢、尾音拖长，表现不舍与迷茫。

## 六、旁白与营销分支

- **旁白同样用公式**：`旁白说：「…」音色：{年龄感+性别+声线质感+语速+情绪底色}`。品牌旁白先定位：沉稳信任型（中年声、音高偏低、语速适中、咬字清楚）/ 活力种草型（青年声、明亮、语速稍快、语气上扬）。
- **营销口播/CTA**：口语化自然（同 P3 营销文案规则）；**重音落在利益点和行动词**（'今天下单''立省'按词加重）；CTA 句尾音干脆不拖。
- **夸张反应豁免**（与微表情营销豁免同源）：试吃惊艳类声音可以拔高，但要有**过程**（正常说话→咬下脆响→声音瞬间拔高半个调），不是全程高八度。
- 带货主播/素人体验仍用克制写法（"真实种草感"本来就要克制）。

## 七、P8 模板速用卡（写音色时直接套）

**单句台词**：
```
{角色名}@图片N说：「{台词原文}」音色：{基础音色：素材清单原样复用}。这句话{声音状态}，{重音位置/停顿方式/句尾处理 挑2~4个}。
```
✅ 例：`苏晚@图片2说：「我从来没有相信过你。」音色：女声，24岁，语调克制隐忍，音高偏低，语速偏慢，口音无，声线质感清冷微哑。这句话气息稳定，'从来'两字加重，句尾轻轻收住，不哭喊。`

**多句跨分镜**：每个分镜套单句模板，声音过程按第四节递进链写（前一分镜的情绪落点 = 后一分镜的起点）。

**多人同框**：每个说话者各自带完整基础音色+本句过程，禁止共用。

**写完自查 6 问**：
1. 有没有**裸情绪词音色**？（"愤怒地说"没有声音过程 → 重写）
2. 每句音色是否 = 基础音色 + 本句过程？基础音色与素材清单**一字不差**？
3. 多句有没有**递进**？（每句声音状态不同，方向符合情绪曲线）
4. 多角色**听感拉开**了吗？（盲听能分清谁在说）
5. 标点与声音过程矛盾吗？（写"情绪爆发"台词却全是平静句号 → 回 P3/P5 核对台词标点）
6. **声画同源**吗？（声音颤抖 ↔ 画面强忍微表情）

## 八、声音问题判责树（成片后用户报"声音不对"）

| 现象层 | 定位 |
|---|---|
| Prompt 音色已含基础音色+声音过程，生成仍不对 | 种子/模型问题 → **重抽该 Clip** 即可 |
| Prompt 音色只写了身份/裸情绪词（"青年音色""愤怒地"） | **修 Phase8**：按第七节速用卡重写音色 |
| 素材清单基础音色登记就错（年龄感不对/多角色无差异/缺扩展字段） | **回 Phase4** 改音色描述 → 传导更新 Phase8 全部相关台词 |
| 台词本身节奏不对（该爆发处全是句号/该停顿处无标点） | **回 Phase3/Phase5** 改台词标点 |

> 与 `rules-and-fallback.md` 回退表 C 联动。⚠️ 改台词标点会改变字数 → 必须重过 Phase5.2 计时与闸 4（字数含标点 ÷5）。

## 总结：AI 声音控制的核心不是情绪，而是结构

- 不要只告诉 AI 角色是谁，要告诉 AI 角色**怎么说话**。
- 不要只写"愤怒""悲伤""激动"，要写**音量、语速、停顿、气息、重音、尾音**。
- 不要只写一整段情绪，要把多句台词**按时间顺序拆开**，让声音跟着情绪变化。
- 三方法：①声音公式控**音色**（年龄感+声线质感+语速+情绪底色+发声习惯）；②声音过程控**语气**（把"他很生气"改成"他说话时音量变高、咬字变重、句尾短促"）；③标点符号控**语速**（逗号减速、省略号停顿、感叹号爆发、跨镜承载制造段落停顿）。
